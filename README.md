# LGTV UART component for ESPHome

ESPHome custom component for DIY LGTV controller using RS232 interface

### Hardware
- Wemos D1 MINI
- MAX3232

```
Wemos D1 MINI : MAX3232
5V: VCC
G: GND
D3: TXD
D4: RXD
```
